How to Create a Simple HTML and PHP Form

Step 1: HTML Form

Create an HTML form with the necessary input fields. Here's an example:

          <form method="POST" action="process_form.php">
            <div>
              <label for="name">Name:</label>
              <input type="text" id="name" name="name" required>
            </div>
            <div>
              <label for="email">Email:</label>
              <input type="text" id="email" name="email" required>
            </div>
            <div>
              <input type="submit" value="Submit">
            </div>
          </form>

Step 2: PHP Form Handling

Create a PHP file named "process_form.php" to handle the form submission. Here's an example:

          <?php
          if ($_SERVER["REQUEST_METHOD"] == "POST") {
            $name = $_POST["name"];
            $email = $_POST["email"];

            // Perform necessary actions with the form data
            // For example, send an email or store the data in a database

            // Display a success message
            echo "Form submitted successfully!";
          }
          ?>
