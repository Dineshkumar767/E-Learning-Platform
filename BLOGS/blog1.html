Arrays and Their Manipulation

JavaScript arrays are powerful data structures that allow you to store and manipulate multiple values in a single variable. They are widely used in web development and provide various methods for manipulation.

Array Creation

An array can be created using the array literal syntax or the Array constructor:

let myArray = [1, 2, 3, 4, 5]; // Using array literal
let myArray = new Array(1, 2, 3, 4, 5); // Using Array constructor

Accessing Array Elements

You can access individual elements of an array using their zero-based index:

let myArray = [1, 2, 3, 4, 5];
console.log(myArray[0]); // Output: 1
console.log(myArray[2]); // Output: 3

Modifying Array Elements

Array elements can be modified by assigning new values to specific indices:

let myArray = [1, 2, 3, 4, 5];
myArray[2] = 10;
console.log(myArray); // Output: [1, 2, 10, 4, 5]

Array Length

The length property returns the number of elements in an array:

let myArray = [1, 2, 3, 4, 5];
console.log(myArray.length); // Output: 5

Adding and Removing Array Elements

JavaScript provides methods to add or remove elements from an array:

let myArray = [1, 2, 3];
myArray.push(4); // Add element to the end
console.log(myArray); // Output: [1, 2, 3, 4]

let removedElement = myArray.pop(); // Remove last element
console.log(removedElement); // Output: 4

myArray.unshift(0); // Add element to the beginning
console.log(myArray); // Output: [0, 1, 2, 3]

let removedFirstElement = myArray.shift(); // Remove first element
console.log(removedFirstElement); // Output: 0

Slicing and Splicing Arrays

The slice method extracts a portion of an array into a new array:

let myArray = [1, 2, 3, 4, 5];
let newArray = myArray.slice(1, 4);
console.log(newArray); // Output: [2, 3, 4]

The splice method can be used to add, remove, or replace elements in an array:

let myArray = [1, 2, 3, 4, 5];
myArray.splice(2, 1); // Remove one element at index 2
console.log(myArray); // Output: [1, 2, 4]

myArray.splice(1, 0, 6, 7); // Add elements at index 1
console.log(myArray); // Output: [1, 6, 7, 2, 4]

myArray.splice(3, 1, 8, 9); // Replace one element at index 3
console.log(myArray); // Output: [1, 6, 7, 8, 9, 4]

Array Iteration

Arrays can be iterated using loops or higher-order methods like forEach, map, filter, and reduce:

let myArray = [1, 2, 3, 4, 5];

// Using for loop
for (let i = 0; i < myArray.length; i++) {
  console.log(myArray[i]);
}

// Using forEach method
myArray.forEach(function(element) {
  console.log(element);
});

// Using map method
let doubledArray = myArray.map(function(element) {
  return element * 2;
});
console.log(doubledArray);

// Using filter method
let filteredArray = myArray.filter(function(element) {
  return element > 2;
});
console.log(filteredArray);

// Using reduce method
let sum = myArray.reduce(function(acc, curr) {
  return acc + curr;
}, 0);
console.log(sum);
