How Functions Work in JavaScript

Introduction to Functions

A function is a block of reusable code that performs a specific task. It allows you to organize your code, make it more modular, and avoid repetition. Here's an example of a simple function in JavaScript:

          function greet() {
            console.log("Hello, World!");
          }

          // Call the function
          greet();

Function Parameters

Functions can accept parameters, which are values passed to the function for it to work with. Here's an example:

          function greet(name) {
            console.log("Hello, " + name + "!");
          }

          // Call the function with a parameter
          greet("John");

Function Return Values

A function can also return a value using the return statement. Here's an example:

          function add(a, b) {
            return a + b;
          }

          // Call the function and store the returned value
          var result = add(2, 3);
          console.log(result); // Output: 5
