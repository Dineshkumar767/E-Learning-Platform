HTML and CSS

HTML (Hypertext Markup Language) and CSS (Cascading Style Sheets) are essential technologies for building websites. HTML provides the structure and content, while CSS controls the presentation and layout.

HTML:

HTML is the standard markup language used to create webpages. It consists of a set of tags and elements that define the structure and hierarchy of content on a webpage. HTML tags can include headings, paragraphs, links, images, lists, tables, forms, and more.

CSS:

CSS is a stylesheet language that works alongside HTML. It allows you to style and format the HTML elements, controlling aspects such as colors, fonts, sizes, layout, and positioning. CSS provides a way to create visually appealing and consistent designs across webpages.

Key Features of HTML and CSS:

  • HTML:
    • Defines the structure and hierarchy of webpage content
    • Uses tags and attributes to mark up different elements
    • Supports multimedia elements, forms, tables, and more
    • Allows linking to external resources, such as stylesheets and scripts
  • CSS:
    • Controls the visual presentation of HTML elements
    • Offers a wide range of styling options, including colors, fonts, borders, backgrounds, and more
    • Enables responsive design for adapting to different screen sizes and devices
    • Allows the use of selectors to target specific elements for styling
    • Supports advanced features like animations and transitions

Sample HTML Code:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Webpage</title>
</head>
<body>
  <h1>Welcome to My Webpage</h1>
  <p>This is a sample paragraph.</p>
</body>
</html>

Sample CSS Code:

body {
  font-family: Arial, sans-serif;
  background-color: #f4f4f4;
}

h1 {
  color: #0066cc;
}

p {
  font-size: 16px;
  color: #333333;
}
